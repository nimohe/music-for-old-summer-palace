
# Zenith Music Player - 本地运行指南

这是一个基于 React + TypeScript + Vite 构建的极简音乐播放器。

## 操作步骤

### 1. 环境准备
确保你的电脑上已经安装了 [Node.js](https://nodejs.org/) (建议使用 LTS 版本)。

### 2. 项目初始化
1. 创建一个新的文件夹。
2. 将项目中的所有文件复制到该文件夹中。

### 3. 安装依赖
打开终端，进入该文件夹，运行：

```bash
npm install
```

### 4. 启动开发服务器
运行以下命令启动：

```bash
npm run dev
```

### 5. 访问项目
启动后，终端会显示：
- **Local**: `http://localhost:5173`
- **Network**: `http://192.168.x.x:5173` (你的局域网 IP)

你可以在电脑浏览器使用 Local 地址，在手机浏览器使用 Network 地址进行同步测试。

## 主要功能
- **胶片旋转播放**：点击左侧封面或下方播放按钮开始/暂停。
- **歌词同步滚动**：右侧歌词随进度高亮。
- **响应式设计**：移动端支持标题跑马灯效果及音量控制切换。
